Dúvidas rápidas sobre o WorkBalance+

O que é o WorkBalance+?

O WorkBalance+ é uma plataforma acadêmica desenvolvida para analisar humor, produtividade e clima organizacional, voltada para estudos e prototipagem.

O sistema armazena dados reais?

Não. Todo o processamento é local e sem banco de dados externo — ideal para fins educacionais.

Como o Mood Score é calculado?

Ele é calculado pela média entre o humor diário e a produtividade informada pelo usuário.

O WorkBalance+ pode ser usado em empresas reais?

Sim. Com integração a banco de dados e autenticação, pode ser facilmente adaptado para uso real.

O que o dashboard exibe?

Ele mostra médias, comparativos, gráficos de comportamento e tendências ao longo do tempo.
